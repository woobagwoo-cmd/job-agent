💡 두 항목 모두 선택사항입니다.

입력하지 않아도 챗봇 이력서 작성과 공고 매칭은 정상 이용 가능합니다.

  • Gemini API Key — 없으면 기본 템플릿으로 이력서가 생성됩니다. 있으면 AI가 이력서를 더 세련되게 다듬어 줍니다.
  • 메일 앱 비밀번호 — 없으면 이메일 발송 기능만 비활성화됩니다.

키를 입력하지 않으면 기본 템플릿으로 이력서가 생성됩니다. 더 정교한 이력서를 원하시면 Google AI Studio에서 무료 발급받으세요.

📧 발송 이메일 SMTP 설정 선택

Gmail, Naver, Daum/Kakao, Outlook/Hotmail, Nate 메일을 지원합니다. 주소의 @도메인을 보고 발송 서버를 자동으로 선택합니다.

일반 로그인 비밀번호가 아닌 앱 비밀번호가 필요합니다. Gmail: Google 앱 비밀번호 발급 / Naver: 메일 환경설정 → POP3/IMAP → 앱 비밀번호 발급
